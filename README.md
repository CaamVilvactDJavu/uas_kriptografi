## Cara menjalankan

1. Git Clone GitHub berikut
```
git clone https://github.com/CaamVilvactDJavu/uas_kriptografi.git
```

2. Masuk ke dalam folder uas_kriptografi
```
cd uas_kriptografi
```

3. Install libraries
```
pip install -r requirements.txt
```

4. Jalankan program
```
python main.py
```
